/* Gaismas AUTO–OFP Autors: [Tavs vārds] Datums: 2026 Funkcija: Ja PIR sensors nokustību – ieslēdz gaismu (LED); pēc 10 sek (maināms) izslēdz, ja nav kustības. */ const int PIR_PIN = 2; // PIR sensors const int RELAY_PIN = 3; // Relejs (vai LED) const int MANUAL_PIN = 4; // Manuālā poga (HIGH = ieslēgt) const long DELAY_MS = 10000; // 10 sekundes (reālam iestatīt 300000 = 5min) bool gaismaIeslegta = false; unsigned long pedejasKustibasLaiks = 0; void setup() { pinMode(PIR_PIN, INPUT); pinMode(RELAY_PIN, OUTPUT); pinMode(MANUAL_PIN, INPUT_PULLUP); digitalWrite(RELAY_PIN, LOW); // sākumā gaisma izslēgta Serial.begin(9600); } void loop() { bool kustiba = digitalRead(PIR_PIN) == HIGH; bool manualaIzsl = digitalRead(MANUAL_PIN) == LOW; // poga nospiesta // 1. Manuālā pārrakstīšana if (manualaIzsl) { if (!gaismaIeslegta) { digitalWrite(RELAY_PIN, HIGH); gaismaIeslegta = true; Serial.println("Manuāli ieslēgts"); } pedejasKustibasLaiks = millis(); // atiestata taimeri delay(200); // atbrīvo no pogas dubultspiediena return; } // 2. Automātiskais režīms if (kustiba) { pedejasKustibasLaiks = millis(); if (!gaismaIeslegta) { digitalWrite(RELAY_PIN, HIGH); gaismaIeslegta = true; Serial.println("Kustība – gaisma IESLĒGTA"); } } // 3. Pārbaudīt, vai laiks izslēgt if (gaismaIeslegta && (millis() - pedejasKustibasLaiks >= DELAY_MS)) { digitalWrite(RELAY_PIN, LOW); gaismaIeslegta = false; Serial.println("Bez kustības – gaisma IZSLĒGTA"); } delay(100); // maza pauze stabilitātei }
